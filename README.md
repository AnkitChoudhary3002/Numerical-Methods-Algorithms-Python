# Numerical-Methods-Algorithms-Python
Numerical Methods Covered:  Root Finding: Bisection, Regula-Falsi, Secant, Newton–Raphson, Chebyshev.  Linear Systems: Jacobi, Gauss–Seidel, LU Decomposition, Gaussian Elimination.  Interpolation: Newton's Divided Difference, Lagrange Interpolation.  Differential Equations: Euler’s Method, Heun’s Method, Runge–Kutta (including RK4).
